# 🎮 Python Mini-Projects

Welcome to my mini-projects collection! This repository is a fun space where I build interactive console games and utilities in Python to practice fundamental programming logic.

---

## 🛠️ Projects Included

### 1. 🎯 Number Guessing Game (`guessing_game.py`)
* An interactive console game where the user guesses a randomly generated secret number.
* Tracks the number of attempts and guides the player with "too high" or "too low" hints.

### 2. 🪨 Rock, Paper, Scissors (`rock_paper_scissors.py`)
* A classic game of Rock, Paper, Scissors played against the computer.
* Implements random selection logic and determines the winner based on game rules.
